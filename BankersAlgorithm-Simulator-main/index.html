How to Use

  1. Enter the number of processes and resource types
  2. Click "Create Tables" to generate input forms
  3. Fill in the Total Instances table with total available resources
  4. Fill in the Allocation Matrix with currently allocated resources
  5. Fill in the Maximum Matrix with maximum resource needs
  6. Click "Calculate Need Matrix" to compute Need = Maximum - Allocation
  7. Click "Calculate Available Resources" to determine remaining resources
  8. Click "Find Safe Sequence" to determine if the system is safe
